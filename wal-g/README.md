# WAL-G for PostgreSQL: Cloud-Native Backup & Recovery

This directory contains technical guides and references for WAL-G, the archival and recovery tool for PostgreSQL designed for cloud object storage.

## Contained Documents

*   **[WAL-G Internals & Operations Report (PDF)](file:///home/josef/github.com/josmac69/postgresql_support_docs/wal-g/WAL-G_Internals__A_Deep_Technical_Report_for_PostgreSQL_Specialists.pdf)**: Deep-dive analysis of WAL-G's architecture, including parallel block-level compression (lz4, zstd), background uploaders, delta-backup mechanisms, and cloud storage integrations (AWS S3, GCP GCS, Azure Blob).
