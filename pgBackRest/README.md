# pgBackRest Architecture & Operations

This directory contains technical guides and references for pgBackRest, a high-performance, parallel backup and restore tool for PostgreSQL.

## Contained Documents

*   **[pgBackRest Architecture Deep-Dive (PDF)](file:///home/josef/github.com/josmac69/postgresql_support_docs/pgBackRest/pgBackRest_Architecture_Deep_Dive.pdf)**: Comprehensive analysis of pgBackRest's multi-process design, TLS secure repositories, page-level delta backups, block-level checksum validation, and PITR (point-in-time recovery) procedures.
