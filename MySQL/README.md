# MySQL & MariaDB Reference Guides

This directory contains technical comparison manuals and guides about MySQL/MariaDB architectures, target translations, and replication consensus models.

## Contained Documents

*   **[MySQL Since 5.7: A Technical Report for PostgreSQL Specialists](file:///home/josef/github.com/josmac69/postgresql_support_docs/MySQL/MySQL_Since_5.7-_A_Comprehensive_Technical_Report_for_PostgreSQL_Specialists.md)**: A detailed mapping of concepts between MySQL/InnoDB and PostgreSQL, comparing locking models, MVCC visibility, transaction isolation levels, indexing, replication, and query execution optimizations.
*   **[Paxos Consensus Protocol in MySQL](file:///home/josef/github.com/josmac69/postgresql_support_docs/MySQL/paxos_protocol.md)**: Technical notes on how Group Replication in MySQL uses the Paxos consensus protocol to achieve distributed consensus and split-brain fencing.
